Genera contraseñas para el sistema de registro vehicular

Presiona "Generar" para crear una contraseña

Instrucciones:

  1. Genera una contraseña con el botón "Generar"
  2. Copia la contraseña con "Copiar"
  3. Comparte la contraseña con quien necesita registrar un vehículo
  4. La contraseña anterior se invalidará automáticamente
